========================================
ALTERAÇÕES REALIZADAS
========================================

1. INTERFACE DO USUÁRIO
   ✅ Atualizada exibição da versão de "Minecraft 1.12.2" para "Minecraft 1.20.1"
   ✅ Mantida compatibilidade com o design neon existente

2. CAMINHOS DE ARQUIVOS
   ✅ Java Runtime: Atualizado de "jre-legacy" para "java-runtime-gamma"
   ✅ Versão Forge: Atualizada de "1.12.2-forge-14.23.5.2860" para "1.20.1-forge-47.2.0"
   ✅ Diretórios de natives e bibliotecas atualizados

3. BIBLIOTECAS E DEPENDÊNCIAS
   ✅ Lista completa de JARs atualizada para Minecraft 1.20.1:
       - Forge 1.20.1-47.2.0
       - LWJGL 3.3.1 (atualizado de 2.9.4)
       - Log4j 2.19.0 (atualizado de 2.15.0)
       - Netty 4.1.77.Final (atualizado de 4.1.9.Final)
       - ASM 9.5 (atualizado de 5.2)
       - Guava 31.1-jre (atualizado de 21.0)
       - E muitas outras dependências modernas

4. CONFIGURAÇÕES DE JAVA
   ✅ JVM Arguments otimizados para Minecraft 1.20.1:
       - Removido: ConcMarkSweepGC (descontinuado)
       - Adicionado: G1GC com configurações otimizadas
       - Adicionado: -Dlog4j2.formatMsgNoLookups=true (segurança)
       - Configurações G1GC para melhor performance

5. LAUNCHER CLASS
   ✅ Main Class atualizada:
       - De: "net.minecraft.launchwrapper.Launch"
       - Para: "cpw.mods.bootstraplauncher.BootstrapLauncher"

6. ARGUMENTOS DE LANÇAMENTO
   ✅ Novos argumentos necessários para 1.20.1:
       - --clientId
       - --xuid
       - --launchTarget forgeclient
       - Asset Index atualizado de "1.12" para "5"

🎮 MINECRAFT:

   - Assets do Minecraft 1.20.1
